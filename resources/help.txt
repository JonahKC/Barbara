**Here's What I Can do!**
`/credits` - Shows the credits of the bot.
`/help` - Shows this message.
`/reaction [add/remove] [emoji]` - Add or remove an emoji reaction to a message.
`/say` - Send a message as Barbara! If you're not an admin, it will be visible that you send this message.
`/pickup` - Sends a pickup line from a hand-curated list of over 400!
ADMIN_ONLY
`/admin [add/remove] [user/role]` - Add or remove a user or role from the admin list.
`/admin list [users/roles/all]` - List all users or roles in the admin list.
`/config read [key]` - Read a config value.
`/config set [key] [value]` - Set a config value.
`/config reset [key]` - Reset a config value to its default.
`/config append [array] [value]` - Append a value to a config array.
`/config remove [array] [value]` - Remove a value from a config array.
`/meesedetect [true/false]` - Enable or disable meese detection/censorship. Alias for /config set nomees [true/false]
